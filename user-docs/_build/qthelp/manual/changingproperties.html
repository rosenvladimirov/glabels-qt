Changing Basic Project Properties

To change the properties of a project, choose File ➡ Properties or click on Properties in the toolbar on the left of the main window. The Properties view will be shown.

Here you see the properties of the label you are working on:

The Product panel shows some label data, like vendor, part number, description and so on. From here, you can switch to another product by clicking on Change product. The window gLabels – Select Product opens, and you can choose from some recently used products, and by clicking on the tab Search all on the left, the whole template collection will be shown.

If you like to change the orientation only, click on the drop-down list under Orientation. Now you can choose between Horizontal orientation and Vertical orientation. Changing the orientation doesn’t rotate the preview on the right; only the arrow (captioned with Up) shows the new direction.

Note

By changing the orientation, you might lose some of your work because the objects you’ve already placed no longer fit. Only the template will be rotated, not the objects on it.
