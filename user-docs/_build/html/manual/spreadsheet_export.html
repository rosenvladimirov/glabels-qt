Export CSV files from various spreadsheet applications¶

LibreOffice Calc¶

Choose File ➡ Save as…. In the Filter box, select Text CSV. (Optional) Set the field options for the Text CSV file. Select Edit filter settings. In the Export of text files dialog, select the options that you want. Click OK and then Save.

Calligra Sheets¶

Choose File ➡ Export… or File ➡ Save As…. The file saver dialog shows the current filename, currently probably with an .ods extension (the default in Calligra Sheets). Open the File type drop-down list and choose CSV document. The filename extension changes automatically to .csv. Then click on Save.

Google Sheets¶

Choose File ➡ Download ➡ Comma Separated Values (CSV). The file will be downloaded automatically into your configured download folder. The original filename will be suffixed with * - SheetX* where X is the number of the sheet (should usually be irrelevant for our purposes).

Gnumeric¶

Currently (2026-01), gnumeric seems to have no options to export .csv files. Please correct me if that is indeed possible.
