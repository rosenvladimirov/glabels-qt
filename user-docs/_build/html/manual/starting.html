Starting gLabels¶

To start glabels, you have a few options:

From the Application Menu¶

KDE Plasma¶

Open the Application Menu by clicking on it, or press Alt+F1. The search field is already active. Type glabels and click on the glabels entry. Or go to Office ➡ glabels Label Designer 4 and click to open it.

GNOME¶

Open the Activities Overview by clicking on the icon in the top left corner of the screen. Type glabels and click on the entry found.

Alternatively, in most desktop environments (and maybe in most window managers, too) you can open a quick start row using Alt+F2. Once it appears, type glabels and click on the entry found.

From the Command Line¶

Open a terminal window and type glabels-qt. For command line options, refer to the man page (to open with man glabels-qt)
